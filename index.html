Create Package

Fill in the package and its connectors. On save, the data appears immediately in the app and is committed to the repo via the GitHub API.

GitHub access

Paste a fine-grained personal access token with Repository access: this repo and Contents: Read and write. Saved on this device (localStorage) so you only enter it once; it is never committed. Use Clear on shared machines.

Connectors

Add a card per connector with its full connector, contact, tooling and accessory details. Only Connector ID and PCB part number are required; leave the rest blank if unknown.
